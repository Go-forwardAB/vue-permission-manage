## 项目介绍

该项目为权限管理后台管理系统基础模板，包含：

- Vue 3 + TypeScript 支持
- Element Plus UI 框架及图标库
- Pinia 轻量状态管理
- Vue Router 4 路由管理
- Vue I18n 国际化支持
- Unocss 和 Tailwindcss 双重原子化 CSS 解决方案
- 自动导入 API 和组件，提升开发效率
- ESLint + Prettier 代码规范和格式化
- Husky + lint-staged 代码提交检查
- 支持热更新的开发环境和生产构建优化
